# Task ID: 1
# Title: PostHog Foundation Setup and Configuration
# Status: pending
# Dependencies: None
# Priority: high
# Description: Set up PostHog JavaScript SDK in the Phoenix/Elixir application with proper environment configuration and initialization
# Details:
1. Add PostHog JavaScript SDK to assets/js/app.js:
```javascript
import posthog from 'posthog-js'

if (window.POSTHOG_API_KEY) {
  posthog.init(window.POSTHOG_API_KEY, {
    api_host: 'https://eu.i.posthog.com',
    loaded: function(posthog) {
      if (window.currentUser) {
        posthog.identify(window.currentUser.id, {
          email: window.currentUser.email,
          user_type: 'authenticated'
        });
      }
    }
  });
  window.posthog = posthog;
}
```
2. Update config/runtime.exs to include POSTHOG_API_KEY environment variable
3. Modify lib/eventasaurus_web/components/layouts/root.html.heex to include PostHog configuration script tag that exposes API key and current user to JavaScript
4. Add PostHog npm package to package.json dependencies

# Test Strategy:
Verify PostHog initialization by checking browser console for successful connection, test user identification for both authenticated and anonymous users, confirm environment variables are properly loaded
